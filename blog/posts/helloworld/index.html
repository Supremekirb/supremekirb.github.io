This is a test blarg post to test my blarg. I mean blog. Using a very minimal and sort of jank 11ty setup that I cobbled together.

Now What?

I'll write stuff here, whenever I happen to feel like it. What kind of stuff? We can join hands and skip into the sunset as we find out together.

Well, that's not really true. Have you ever watched Skawo on youtube? They do techincal deep-dives into strange bugs and oddities in game code. I absolutely adore that kind of thing, and I'd like to do it for the games I love (and affectionately tear limb from limb). Maybe. We'll see.

You Can Just Add Headers, Nobody's Stopping You

As for this stinky little site, I have Plans. I'd like to make lots of little games to hone my skills, and then I can put them on the web. Isn't that nice? It sure beats sending .zip files around. Sadly, though, Neocities doesn't let you do that sort of thing without paying, which I suppose is fair enough. So it's either that, or I get a domain of my own and piggyback off GitHub or something for hosting.

I should also do things like add a gallery for art of Bit, improve the projects page, and make more online tools. Stuff like an EB damage calculator, various converters like BPM-to-NSPC-tempo, 8.8 conversion, and anything else that comes to mind.

I'd like to improve this blog too. Since I've already got a tag system set up, I should figure out how to let you filter by that.
