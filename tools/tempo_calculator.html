Tempo Calculator < Back to Tools

This tool converts between beats-per-minute (BPM) and N-SPC tempo values (that you input in EBMusEd).
The formula for BPM → N-SPC is 12288 / (60000 / bpm).
The formula for N-SPC → BPM is 60000 / (12288 / nspc).
As you change the input in one box, the other will update. All input values are in decimal.

BPM tempo



Integer: 100

N-SPC tempo



Integer: 20
Hexadecimal: 0x14
Tempo instruction: [E7 14]
