EBME

The primary map editor for EarthBound, made to replace the previous editor which was nearly 20 years old. It's written in Python using Qt via PySide6. It reads and writes CoilSnake projects.

Features:
  • Complete undo / redo support
  • Editing data of objects within the program
  • Interactive png2fts integration (convert images to map data)
  • Collision painting
  • Advanced tools for manipulating palettes
  • Mario Paint minigame
GitHub Starmen.net

EarthBound Trial Tower

Made for a 2-week hacking jam, EB Trial Tower turns the game into a complete rougelike; including 100 floors of battles, over 30 types of upgrades, and the bass from Gimmick! for NES. I've continued to work on it since the initial release.

Starmen.net

EarthBound Beginnings Remake

I've been working on this project as a programmer and scripter since early 2023, as well as occasional small contributions of graphics and music. Things I've done include the reflections in the castle, the train system, the shops, several cutscenes, the walk-off animations at the edge of areas, and more.

Official website Starmen.net Discord

EarthGround

This was made for the first PK Hack Jank Jam. It's a rewrite of the collision and player movement code in EarthBound to turn the game into a 2D platformer. Southwards is downwards, and you can use the Y button to jump. Complete with wall kicks, it's possible to beat the entire game in this strange way.

Additionally, the code behind it is distributed in a script you can include in your own EB hacks!

Starmen.net GitHub

MOTHER 3 prologue SNES demake

A hack of EarthBound which turns it into a highly accurate recreation of MOTHER 3's prologue sequence. This was made for the 17th anniversary MOTHER 3 funfest on Starmen.net. Currently, it's possible to play up until the area with the Dragos. I plan on finishing it someday.

Starmen.net Demonstration video

Joplin To-do View

A plugin for the note-taking app Joplin which displays all your to-do items in a list. They're sorted into categories based on when they're due.
